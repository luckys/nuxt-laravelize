# nuxt-laravelize

Nuxt module that brings Laravel-inspired backend primitives to Nuxt and Nitro.

## Current bootstrap scope

- request-scoped service container wiring
- Nitro server plugin registration
- runtime container composables
- playground and module fixtures aligned with `nuxt-laravelize`

## Local development

```bash
pnpm install
pnpm dev:prepare
pnpm dev
```

## Quality checks

```bash
pnpm lint
pnpm test
pnpm typecheck
```
